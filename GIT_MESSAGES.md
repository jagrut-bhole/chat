| Type         | Meaning                                  | Example                                    |
| ------------ | ---------------------------------------- | ------------------------------------------ |
| **feat**     | New feature added                        | `feat: add Google login authentication`    |
| **fix**      | Bug fixed                                | `fix: resolve crash on signup form submit` |
| **chore**    | Maintenance work (no feature/bug)        | `chore: update dependencies`               |
| **docs**     | Documentation changes                    | `docs: update README installation steps`   |
| **style**    | Formatting/UI only (no logic change)     | `style: improve button spacing in navbar`  |
| **refactor** | Code improvement without behavior change | `refactor: simplify user validation logic` |
| **test**     | Add or update tests                      | `test: add unit tests for auth controller` |
| **perf**     | Performance improvement                  | `perf: optimize database query for groups` |
| **build**    | Build system or config change            | `build: configure webpack for production`  |
| **ci**       | CI/CD related changes                    | `ci: add GitHub Actions workflow`          |
| **revert**   | Revert a previous commit                 | `revert: remove Google login`              |
